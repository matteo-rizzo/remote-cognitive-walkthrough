ReCoWa

An online tool for conducting Remote Cognitive Walkthroughs

How does it work?

The cognitive walkthrough is conducted remotely via a videoconferencing tool (e.g., Zoom). One of the two approaches can be followed:

  1. The user shares his/her screen and follows the instructions of the designer. The user will be shown the main panel and subsequently asked to change panel based on part of the UI he/she wants to interact with.
  2. The user shares his/her screen and either asks the user how he/she would interact with the UI or to use annotation tools to highlight the part of the interaction he/she wants to interact with. The user will be shown the main panel and subsequently the designer will change it based on his/her interaction.
